## 🔄 メンテナンス方針

本マップは **手動キュレーション**（各文書に「いつ読むか」の1行説明を付ける）であり、自動生成はしません。

- **対象**: `doc/` 配下の `.md`（ただし `doc/DD/` 配下は除外＝アーカイブ `doc/DD/archived/` を含む）。
- **更新タイミング**: `doc/`（`doc/DD/` を除く）に文書を **追加・移動・改名・削除** したら、**同じ作業の中で本ファイルを更新する**（規約は [CLAUDE.md](../CLAUDE.md) の「ドキュメント索引（INDEX_MAP）の維持」節）。
- **完了前ゲート**: 上記の変更を伴う作業では、報告前に `bash scripts/doc-index-check.sh --strict` を実行し **exit 0** を確認する。
- **ドリフト検知**: 索引漏れ・リンク切れは次のコマンドで検出できる（読み取り専用・破壊なし。「索引に未登録の doc ファイル」と「本マップ内のリンク切れ」を一覧表示）。

  ```bash
  bash scripts/doc-index-check.sh          # レポートのみ（exit 0）
  bash scripts/doc-index-check.sh --strict # 差分があれば exit 1
  ```
