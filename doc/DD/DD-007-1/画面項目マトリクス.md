## S-04 会議開始プリフライト [S-04](../../mock/html/S-04_preflight.html)
| 項目 | 入力/表示 | 永続化先 |
|------|----------|---------|

## S-05 リアルタイム議事録（会議中・中核） [S-05](../../mock/html/S-05_realtime.html)
| 項目 | 入力/表示 | 永続化先 |
|------|----------|---------|
| 確定セグメント（speaker/t/text） | 表示(主役) | `timeline_elements`（kind=ai_transcription, text_raw, t_ms, speaker_id, seq） |
| LLM整形（薄字・showRefined） | 表示(追上) | `timeline_elements.text_refined` / `is_refined` |
| unrefined バッジ | 表示 | `is_refined=0` から導出 |
| 話者名クリック→参加者から確定（一括置換） | 操作 | `speaker_mappings.confirmed_name`/`confirmed_participant_id` |
| AI推測話者名 | 表示 | `speaker_mappings.ai_guess_name` |
| 人間メモ入力（Enter挿入） | 入力 | `timeline_elements`（kind=human_memo, text_raw, t_ms, seq） |
| 経過時間 / 遅延ゲージ / drop数 / bypass | 表示 | 永続化なし（揮発メトリクス） |
| 「会議を終了」→S-06 | 操作 | `meetings.status='generating'` |
| 右ドロワー（アジェンダ/参加者/用語） | 表示 | `meetings.agenda`/`participants`/`vocabularies` |

## 🔬 孤児チェック結果（書き込み元の無い表示項目）

| 表示項目 | 検出 | 対応 |
|----------|------|------|
| S-03「batch: モデル名」 | 要件4表に無い | → `meetings.batch_model` を**新規追加** |
| S-07「所要 4分12秒」 | 要件4表に無い | → `meetings.generation_seconds` を**新規追加**（or actual差から導出も可、表示安定のため列保持） |
| S-08「音声を保存」実体 | 要件4表に無い | → `meetings.audio_path` を**新規追加** |
| S-05 確定セグメント全般 | 要件4表に**timeline表が無い** | → `timeline_elements` を**新規追加**（中核） |
| S-05 話者確定/AI推測 | 要件4表に**mapping表が無い** | → `speaker_mappings` を**新規追加**（中核） |
| S-08 設定全般 | 要件4表に無い | → `app_settings` を**新規追加** |

→ 孤児（行き先不明の表示）は0件に解消。新規6項目（3テーブル＋3カラム）を抽出。
