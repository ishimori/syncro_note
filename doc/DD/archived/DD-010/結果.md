# DD-010 結果: 逐次STT 実測（medium/4スレッド/最大10秒）

音源 120秒 を 16 チャンク（4.9〜10.0秒）に区切って逐次文字起こし。

| 指標 | 値 | 意味 |
|---|---|---|
| 処理の速さ(RTF) | 0.527 | 間に合う（処理が音声より速い） |
| 体感遅延(平均) | 4.0秒 | 話し終えてから文字が出るまで（平均） |
| 体感遅延(最大) | 4.6秒 | 同（最も遅いとき） |
| 取りこぼし | 15.5% | 一括時からの変化（小さいほど良い） |
| 文字誤り(逐次) | 26.4% | 台本との誤り（用語含む） |
| 文字誤り(一括) | 26.9% | 参考: 全体一括時の誤り |
| └ 区切りの上乗せ | -0.5% | 逐次−一括（区切り由来の悪化分） |

逐次STTの合計処理時間 63.3秒（音声 120秒）。
